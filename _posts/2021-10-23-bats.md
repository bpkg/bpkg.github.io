## Usage
Write tests into `*.bats` files like:

```
#!/usr/bin/env bats

@test "addition using bc" {
  result="$(echo 2+2 | bc)"
  [ "$result" -eq 4 ]
}
```

Then run the test:

```
$ bats test.bats

```
